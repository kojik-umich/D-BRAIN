【テストについて】

・ひとまず基本方針としては google test を用いることとします．
　VisualStudio2017導入時にgoogle test のインストールを忘れずに．あとからでも追加できますが．

・テストの方法は，テストしたいプロジェクトを「.exeを作成する」から「静的ライブラリを作成する」に変更する．
　テストプロジェクトをターゲットプロジェクトに設定する．
　で，できます．

・カバレッジを測定するためには，OpenCppCoverage というライブラリをインストールしてください．
　Visual Studioのメニュー："拡張機能と更新プログラム"からプラグインがインストールできます．


【クラスダイアグラム作成の環境構築】
VisualStudioInstaller起動
変更をクリック
個別のコンポーネントタブ
分類：コードツール　から探す
クラスデザイナーをチェック
依存関係の検証をチェック
ダウンロード
